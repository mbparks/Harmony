Harmony

A browser-based chord synthesizer. Field Instrument 006 by M.B. Parks, Green Shoe Studio. Descended from a software emulation of the Telepathic Instruments Orchid.

Mouse

Click a dial to push it (toggle / open menu). Clicks won't change the value unless you actually drag.
Drag any dial vertically to change its value (5px threshold so clicks stay clean).
Scroll the mouse wheel over any dial to step its value (often the easiest with a mouse).

Keyboard, right hand (notes)

A S D F G H J K = white keys, C through C
W E T Y U = black keys

Keyboard, left hand (chord shaping)

Z Dim (tense)   X Min (sad)   C Maj (happy)   V Sus (floating)
B 6 (sweet)   N m7 (bluesy)   M M7 (dreamy)   , 9 (full)
Hover any chord-type or extension button for a one-line mood description.

Beats, Recorder, MIDI

BPM dial: turn or wheel for tempo, push to open the Beats pattern menu.
Key dial: turn or wheel for the chromatic root, push to cycle Off → Major → Minor → Off. The OLED header shows the current key.
Bass dial: turn or wheel to cycle bass instrument sounds, push to open the bass menu. Options: Off, Auto (follows your chord presses), Doubled (doubles the chord an octave down), Hand-played (you play bass with the keys), Bass Only (silences the chord voices).
Chord Shape dial (center, large): turn to change how the chord is arranged. Different settings give the same chord a different feel.
Bass Octave dial (center, small): turn to shift the bass note up or down by whole octaves.
Channel Strip: two stacked rows of compact sliders. MIX (top) sets independent gains for the synth chord voices, the bass voices, and the loaded backing audio file. FX (bottom) sets reverb, delay, chorus, and drive intensities. All directly editable: drag, scroll, or arrow keys. The top-row Master dial controls the final output volume after everything mixes together.
Recorder: one state-aware primary button (RECORD → STOP → PLAY → PAUSE → RESUME) plus Start Over (abandon take), Save WAV (download stereo 16-bit WAV; auto-stops the recording first if you click while still recording), Load File (any browser-decodable audio loops in sync as a backing track). Adjust the loaded track's playback volume with the Backing slider in the MIX strip.
MENU button: push opens the Settings menu (View options plus MIDI Out devices). Push again to close.
In any OLED menu, click a row to commit, or use the dial that opened it. Chord notes go to MIDI channel 1, bass to channel 2, drums to channel 10.
Note: some embedded contexts (sandboxed iframes, certain previews) disable Web MIDI via Permissions-Policy. If the MIDI menu shows "Blocked by host page", save and open the file standalone.

Patches

The two brass Save Patch / Load Patch buttons on the bottom of the chassis read and write a JSON snapshot of every user-controlled setting: sound preset, performance mode, key, transpose, bass mode and preset and voicing, BPM, master volume, chord type buttons, extensions, view, beats pattern, and all four FX intensities. Runtime state (last-played chord, captured audio, loop transport, MIDI device list) is deliberately excluded.
Each save writes a fresh timestamped .json file. Load accepts any file matching the Harmony patch format.

LED rule

An LED above a dial is lit when that dial's primary function is engaged. Key LED lit means Key Mode on. Bass LED lit means Bass on. BPM LED lit means Beats playing.

Architecture notes

Modules: EventBus, State, AudioCore, SynthVoice, SynthEngine, SoundBank, ChordEngine, KeyEngine, PerformanceEngine, EffectsChain, BassEngine, LoopEngine, BeatsEngine, Patch, MidiOut, Display, UI.
Search the source for // EXTEND: to find extension points (additional sounds, FX, performance modes, beats patterns, loop overdub).
